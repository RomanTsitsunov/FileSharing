# Сервис обмена файлами

## Описание
Программа представляет собой приложение, которое позволяет загружать файлы и получать временную ссылку на скачивание

### Frontend
- Стек: Vanilla JS, CSS, HTML
- Форма загрузки файла
- Отображение статуса загрузки
- Отображение ссылки
- Статистика по файлам

### Backend
- Стек: Java 21 (Maven)
- Автоматическое удаление устаревших файлов(устаревшими файлами считать те, которые не скачивались в течение определенного срока, например, 30 дней)
- Генерация уникальной ссылки на скачивание
- Загрузка файлов доступна только авторизованным пользователям

## Запуск
1. Клонирование проекта:
   ```bash
   git clone https://github.com/RomanTsitsunov/FileSharing
   cd FileSharing
   ```
2. Запуск проекта:
   ```bash
   mvn clean compile exec:java -Dexec.mainClass="doczilla.filesharing.Main"
   ```
   
3. Открыть http://localhost:8080/
